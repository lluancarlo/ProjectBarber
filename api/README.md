# API -  FLASK + SQLAlchemy

app/routes.py - Rotas padrao do sistema (a aplicação deve ser modularizada como no exemplo da pasta "admin").

app/helpers - Deserializers para JSON, codigo re-aproveitado.

app/models/ - Pasta aonde ficaram os models do backend.

app/index.py/ - Rota index
