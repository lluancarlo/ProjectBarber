# ProjectBarber
Projeto de app para barbearias 


## Padrões
Vamo manter padrão pra deixar esssa merda organizada.


Criar uma branch por tarefa, sempre seguindo o padrão feature/'nomedabranch~'.


Manter padrão de formatação de código e styleguides.


Criar um ReadMe com uma doc das tecnologias usadas.


# Back-end está na pasta de api

Para instalar as dependencias basta ter instalado o pipenv e usar o 

> `pipenv install`

Entrar no shell do pipenv com:

> `pipenv shell`

e logo em seguida para rodar o projeto com o comando: 

> `python manage.py runserver`
